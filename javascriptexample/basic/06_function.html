익명 함수(이름이 없는 함수)

        
            function(매개변수){ // 함수선언①
                // 함수 정의
            }

            const 변수 = function(매개변수){ // 함수선언②
                // 함수 정의
            }
        

        → 익명 함수는 이름이 없는 함수이기 때문에, 직접 호출하려면 변수나 매개변수에 저장해야 사용이 가능하다.
        따라서, 선언 즉시 호출하거나 이벤트 처리에 사용되는 경우에 주로 활용된다.

즉시 실행 함수

        
            (function(){
                // 함수 정의
            }());

            (function(매개변수){
                // 함수 정의
            }(인자값));
        
        - 익명 함수의 한 종류로, 함수가 정의되자마자 바로 실행되는 함수이다.

        ★ 즉시 실행 함수 사용 이유
        호출부가 따로 없어 일반적인 함수보다 속도 측면에서 효율적이다.

        전역 변수와의 충돌 방지
        → 즉시 실행 함수 내부는 자체적인 지역 스코프를 가지므로, 전역 변수와 동일한 이름의 변수를 사용해도 충돌하지 않는다.

화살표 함수

        ES6 이후 도입된 간결한 함수 표현식으로, 기존 function 키워드를  =>로 대체하여 작성한다.
        
            작성법 1. 기본 형태([매개변수]) => { 함수 정의 }
                익명 함수   : function(){ }
                화살표 함수 : () => { }

                익명 함수   : function(매개변수){ }
                화살표 함수 : (매개변수) => { }
        

        작성법 2. 매개변수가 "하나"일 경우: 괄호 () 생략 가능
            익명 함수   : function(매개변수) { }
            화살표 함수 : 매개변수 => { }

        작성법 3. 매개변수가 없을 경우 "무조건" () 작성
            화살표 함수 : () => { }

        작성법 4. 함수 정의가 return문 하나일 경우
            function 함수명() { return 값; }
            4-1. return 생략 가능
            4-2. 반환값이 객체일 경우 {}는 코드 블록으로 해석되므로, 객체 리터럴은 ()로 감싸야 함.
