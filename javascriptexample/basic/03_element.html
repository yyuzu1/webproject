DOM(Document Object Model)

        웹 문서(HTML)의 모든 요소를 객체 형식으로 표현한 구조
        → 자바스크립트를 통해 문서 내 특정 요소를 조작하거나 접근할 수 있게 함

DOM을 이용한 HTML 요소 접근 방법(요소 객체 가져오기)

        1. id로 접근하기
        document.getElementById("id속성값");

        2. name으로 접근하기 (복수 요소 반환)
        document.getElementsByName("name속성값");

        3. class로 접근하기 (복수 요소 반환)
        document.getElementsByClassName("class속성값");

        4. 태그명으로 접근하기 (복수 요소 반환)
        document.getElementsByTagName("태그명");

        5. CSS 선택자로 접근하기
        1) 단일 요소 접근
            document.querySelector("CSS 선택자");
            → 선택된 요소가 여러 개인 경우 첫 번째 요소만 반환

        2) 복수 요소 접근
            document.querySelectorAll("CSS 선택자");
